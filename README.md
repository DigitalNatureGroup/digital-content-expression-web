# digital-content-expression-web

このレポジトリは、ディジタルコンテンツ表現演習の制作展のWebページを公開するためのレポジトリです。



## ディレクトリ構造

- `/`
  - 当該年のホームページを格納する
- `/20XX`
  - 20XX年のホームページを格納する



## URL

このレポジトリはGitHub Pagesによってホスティングされています。

GitHub PagesのURLに対してカスタムドメインを適用しています。
